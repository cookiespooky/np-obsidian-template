# Theme Overrides

Эта заметка служебная.

Инструкция:
1. Меняйте только значения в блоке `frontmatter` (между `---` и `---`).
2. Для цветов используйте только `#RGB` или `#RRGGBB`.
3. Если значение ошибочное, оно автоматически игнорируется.
4. При ошибке применяется стандартное значение из `.np/theme/assets/styles.css`.
5. Текст ниже `frontmatter` можно редактировать как угодно: он не участвует в генерации CSS.
