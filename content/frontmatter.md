# Свойства страниц  
  
В начале каждой заметки есть небольшая шапка с настройками страницы.  
  
Она выглядит так:  

```yaml
---
type: article
title: Моя первая статья  
description: Короткое описание страницы  
slug: first-post  
image: /media/image.jpg  
---
```

После неё идёт обычный текст заметки.  
  
Эта шапка нужна, чтобы сайт знал:  
  
- название страницы  
- адрес страницы  
- описание  
- изображение
  
В Obsidian такие свойства называются **frontmatter**.  
  
В шаблоне всё уже настроено — можно просто копировать существующие страницы.
