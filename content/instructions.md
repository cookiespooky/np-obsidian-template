# Сайт из Obsidian - подробная инструкция

Настройка требуется только 1 раз. После этого публикация сайта происходит в 1 клик.

Ваш сайт по умолчанию запускается на GitHub Pages - бесплатном хостинге сайтов от GitHub.

К сайту на GitHub Pages можно подключить свой домен и настроить любой внешний вид.

## Шаг 1. Скопировать шаблон

Перейти на [страницу шаблона](https://github.com/np-obsidian-template) и нажать Use this template → Create a new repository:
![github notepub obsidian template](template.png)
## Шаг 2. Клонировать на свой компьютер

Способ 1 (рекомендуется)  
GitHub Desktop  
  
Способ 2  
терминал:
`git clone https://github.com/owner/repository /путь_к_папке_на_вашем_компьютере`,

где `owner` - ваш логин в GitHub, а `repository` - название созданного репозитория

## Шаг 3. Открыть папку как vault в Obsidian  

По умолчанию тексты лежат в папке `content/`, а медиафайлы - картинки, видео - в папке `/media`. Все редактируется в Obsidian. Для корректной работы страниц, учитывайте [[frontmatter|свойства заметок (frontmatter)]].

## Шаг 4. Настроить публикацию

Установить Git плагин из Community plugins. После установки появится иконка Git плагина, которая открывает панель плагина. Теперь просто нажмите Commit & sync, и GitHub автоматически соберёт и обновит сайт. Публикация занимает около минуты. Процесс можно наблюдать во вкладке Actions вашего репозитория.
