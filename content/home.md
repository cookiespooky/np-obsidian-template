# Этот сайт сделан из Obsidian

Все страницы — обычные заметки.  
Пишете заметки → получаете сайт.

>Без хостинга, сервера, баз данных, CMS, админок и подписок.
>SEO настроено по умолчанию.

## Как это работает

1. Вы пишете заметки в Obsidian
2. Push через Git plugin
3. Notepub собирает сайт из вашего vault

Как заметка превращается в страницу сайта:

![Notepub-demo](/media/demo.mov)

## Подходит для
  
• личного сайта  
• digital garden  
• документации  
• базы знаний  
• SEO-блога
