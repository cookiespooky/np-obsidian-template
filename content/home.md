# Этот сайт сделан из Obsidian

Все страницы — обычные заметки.  
Пишете заметки → получаете сайт.

>Без хостинга, сервера, баз данных, CMS, админок и подписок.
>SEO настроено по умолчанию.

**Подходит для:**  
• личного сайта  
• digital garden  
• документации  
• базы знаний  
• блога

## Как это работает

1. Пишете заметки в Obsidian
2. Делаете push через Git plugin
3. Notepub собирает сайт

**Как заметка превращается в страницу сайта:**

## Свой сайт из Obsidian за 10 минут

Шаблон бесплатный.  
Настраивается один раз.  
После этого сайт публикуется в один клик.

1. Скопировать шаблон
2. Открыть в Obsidian и изменить тексты
3. Опубликовать через Git plugin
