# Почему Notepub

Есть несколько способов публиковать заметки из Obsidian.

## Obsidian Publish

- + простой запуск
- - платная подписка
- - закрытая система
- - интерфейс сайта не кастомизируется

## Статические генераторы (Hugo, Astro)

- + мощные
- - требуют обязательных навыков разработки
- - не работают напрямую с хранилищем Obsidian

## Notepub

+ + работает напрямую с Obsidian  
+ + open source  
+ + бесплатный  
+ + Git workflow  
+ + SEO настроено по умолчанию

Вы пишете заметки → получаете сайт.
